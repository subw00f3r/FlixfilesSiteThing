Uploading Images to Roblox

Follow the steps below to create your own decals:

  1. Open roblox.com and select the Create tab.
  2. Now select the Decals button.
  3. Select the image you wish to upload.
  4. Finaly, choose a name for the decal!

Get your Image ID

There are two ways to get your Image ID

From roblox.com

  1. Select the image you wish to use.
    2. In the URL look for the numbers "roblox.com/library/Numbers/"
  3. Select and copy the numbers by either:
    1. right clicking and selecting copy.
    2. pressing Ctrl and C.

From Roblox Studio

  1. Open roblox studio and select a game
  2. Open the toolbox by selecting: view > Tooblox
    1. If you are using someone else's Decal
      1. Select images from the dropdown menu.
      2. Right click the image you wish to use and select "Copy Asset ID".
    2. If you are using your own Decal
      1. Press the "Inventory" button.
      2. Select images from the dropdown menu.
      3. Right click the image you wish to use and select "Copy Asset ID".

If you need more information, please visit roblox's help center: https://en.help.roblox.com/hc/en-us/articles/203313930-How-to-Make-Decals

Setting up your smartwall

Documentation and examples for the: Static Controller, Slideshow Controller, Emergency Event & Fade Time

Static Controller

The static controller allows you to easily select and display images on your smartwall.

By default your smartwall will come with only one static controller but you can duplicate it to get as many as you wish.

  1. To start, select the smartWall you wish to set up.
  2. In the explorer navigate to: Static_Controller > Screen > Display > Main > Images.
  3. Now select the item you wish to set up.
  4. In the item select "ImageButton" and paste the image ID in the "image" property
  5. Now select the "TextLabel" and give the item a name in the "text" field.

Repeat this with all the items you wish to set up.

If you have more than 8 decals you can duplicate the Static_Controller and repeat the steps above.

If you have any questions or need help you can create a support ticket here: Scada's Discord server

Note: You have to pause the slideshow to use the static controller

Slideshow Controller

The Slideshow Controller lets you automatically cycle trough multiple images.

You can change the amount of slides aswell as how long each image should be displayed.

  1. To start select the SmartWall you wish to set up.
  2. In the exporer navigate to: Slideshow_Controller.
  3. Select the "Slideshow_Config" script.
  4. In the first item rbxassetid:// between "" after image like displayed.
  5. You can also change the time the image is displayed by changing "time". (in seconds)

ATTENTION: Common mistake: if the images do not show up you probably forgot to type rbxassetid:// before the ID.

If you have any questions or need help you can create a support ticket here: Scada's Discord server

Emergency Event

The Emergency Event is a "Bindable Event" that you can fire to display an emergency message on screen.

You can use this code in your own scripts to send the emergency signall.


To interface with the SmartWall emergency input use the following code

emergency = script.Parent.... Enter the path to the emergency bindable event here.

emergency:Fire() This sends the emergency signall.


You can change the emergency image by changing the value in Monitor > Screen > SurfaceGui > Emergency Image

You have to type rbxassetid:// before the ID!!

Fade Time

You can change how long it takes for the old image to fade into the new one.

To do this change the "Fade Time" value. (in seconds)

Monitor > Screen > SurfaceGui > Fade Time
