FAQ's

motion sensor doesn't work.
  1. Make sure sensor and light share same "group" value
  2. Controll group:: By default the sensor is deactivated, a switch can be added to the Control_Group to enable and disable the sensor.
  3. If client wants sensor to be always enabled set Control_Group_On_By_Default to true.
